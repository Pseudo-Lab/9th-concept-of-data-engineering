## 스터디 목표

- **Main Goal**
    - **데이터 엔지니어링** 기초 다지기
- **Sub Goals**
    - 개인 기술 블로그를 통해 학습 내용을 정리하고 공유하기
    - DataCamp의 Data Engineer Certification 취득하기
